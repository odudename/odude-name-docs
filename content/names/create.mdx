# Creating a Name

Getting started with your own Odude Name is simple. Follow these steps to register your unique identity.

## Steps to Create

1. **Navigate to Registration**: Go to the name registration page on the Odude portal.
2. **Search Availability**: Enter your desired name to check if it's available.
   
   ![Search bar availability check](/assets/docs/names/search-placeholder.png)

3. **Select and Register**: If available, proceed to confirm the registration.

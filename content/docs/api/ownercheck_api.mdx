# Owner Check API

The Owner Check API allows you to verify ownership of a project or resource.

## Endpoint
`GET /api/ownercheck`

## Parameters
| Parameter | Type | Description |
| :--- | :--- | :--- |
| `id` | string | The unique ID of the resource. |

## Response
Returns a JSON object indicating ownership status.

```json
{
  "status": "success",
  "is_owner": true,
  "owner_id": "user_123"
}
```
